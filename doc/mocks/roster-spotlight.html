Design notes for the implementor.
  • Roster rows are fixed-height (about 64px including the third line). At N=15 the rail scrolls; the pinned blocks (terminated / notices / topology) never compete with the roster for height.
  • The leading 3px accent rail on each row carries the same semantic as today's .agent-column-accent: idle (transparent), active (tool, muted), stuck (tool ≥5s, full + pulse), thinking (orange, pulse), done (green, muted), error (red).
  • Selection is communicated three ways simultaneously: row background lifts to --bg-0, a 2px right-edge inset glows --accent, and the checkbox fills. Unambiguous at any density.
  • Spotlight pane is data-shape-identical to today's column. Same header, status row, tool-chip row, message stream, input. Reuses every existing component — this is a layout change, not a redesign.
  • Empty spotlight (zero selected) inverts to a centered prompt: "Pick one or more agents from the roster to focus."
  • Topology is anchored bottom-left and never scrolls with roster. Stays the operator's persistent map.
